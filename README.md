# FAST NUCES Portfolio Website

A responsive, high-performance college portfolio website built for FAST-National University of Computer & Emerging Sciences.

## Live Demo



## Tech Stack

- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS v3
- **Routing:** React Router DOM v6
- **Deployment:** Vercel
